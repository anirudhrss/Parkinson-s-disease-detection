Steps to run the code:

Dependencies:

-> Anaconda launcher
-> Jupyter notebook 
-> OpenSmile
-> Python 3.8 and above
-> Dataset (https://zenodo.org/record/2867216#.XeTbN59R2BZ)

Steps:

1. Unzip the .py file.
2. Launch jupyter
3. From file select open drop down
4. Navigate to the .py file in system downloads
5. Load the file
6. Check python version 
7. Install openSMILE 
8. Place the dataset on mentioned location in C:\Users\
9. Run all the cells from the start by selecting run all from Cell drop down.
